
## ✅ Pasos para agregar tu logo:

1. **Prepara tu logo**
   - Formato: PNG con fondo transparente
   - Tamaño: 512x512 píxeles (cuadrado)
   - Nombre: `logo.png` (todo en minúsculas)

2. **Navega a la carpeta**
   ```
   cd C:\Users\MMLAGOS\Documents\MICHAEL\joyeria-proyect\joyeria-front\public
   ```

3. **Copia tu archivo**
   - Pega tu `logo.png` dentro de la carpeta `public`
   - Reemplaza si ya existe uno

4. **Recarga el navegador**
   - Presiona `Ctrl + R` o `F5`
   - ¡Tu logo aparecerá automáticamente! ✨

## ❓ ¿Problemas?

Si el logo no aparece:
1. Verifica que el nombre sea exactamente `logo.png`
2. Asegúrate de que esté en la carpeta `public/`
3. Recarga el navegador con `Ctrl + Shift + R` (recarga completa)
4. Verifica que sea formato PNG

**¡Eso es todo!** Solo copia `logo.png` a la carpeta `public/` y recarga. 🚀
