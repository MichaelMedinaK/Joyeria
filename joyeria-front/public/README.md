# 📁 Carpeta Public

Esta carpeta contiene los archivos estáticos que se sirven directamente.

## 🖼️ Logo

**Para agregar tu logo personalizado:**

1. Coloca tu archivo de logo aquí con el nombre: **`logo.png`**
2. Ruta completa: `joyeria-front/public/logo.png`
3. Tamaño recomendado: 512x512px (cuadrado)
4. Formato: PNG con fondo transparente

El logo aparecerá automáticamente en:
- ✅ Sidebar (esquina superior izquierda)
- ✅ Pantalla de login
- ✅ Favicon del navegador

Si no colocas el logo, se mostrará el emoji 💎 como respaldo.

## 📂 Archivos Estáticos

Puedes colocar aquí cualquier archivo estático:
- Imágenes
- Iconos
- Fuentes locales
- Archivos públicos

Acceso: `http://localhost:3000/nombre-archivo.ext`
